# 🚀 Modern Personal Portfolio

A sleek, modern, and fully responsive personal portfolio website built from scratch. This project showcases my skills in web development, design aesthetics, and interactive user experiences.

## 🌐 Live Demo
Check out the live website here: 
👉 [https://basselmo.github.io/Protfolio-Demo/](https://basselmo.github.io/Protfolio-Demo/)

## ✨ Key Features

* **🌑 Elegant Dark Theme:** A professional dark-mode aesthetic designed for modern readability and visual appeal.
* **📱 Fully Responsive:** Optimized for all devices, including desktops, tablets, and mobile phones.
* **🎭 Smooth Animations:** Engaging UI transitions and interactive elements powered by CSS and JavaScript.
* **📩 Functional Contact Section:** A dedicated area for visitors and potential clients to reach out.
* **🏗️ Clean & Structured Code:** Developed with best practices, making the codebase easy to maintain and scale.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Advanced styling using Flexbox, CSS Grid, and Custom Properties (Variables).
* **JavaScript (Vanilla JS):** Core logic for interactivity, animations, and dynamic UI updates.
